Users' Guide for ALARA:

Analytic and Laplacian Adaptive Radioactivity Analysis

The primary purpose of ALARA is to calculate the induced activation caused by neutron irradiation throughout a nuclear system (including fission reactors, fusion reactors, and accelerators).

This Developers' Guide will describe the source code and structure of ALARA, serving as a guide to those who will continue development of ALARA.

Language and Disclaimer

Although ALARA has been written in C++, the project served as a learning exercise for both C++ and object-oriented programming. Thus, while a great deal of effort was made to write well designed object-oriented code, as of this writing, the author has had no confirmation that this was successful. Nevertheless, it is recommended that all additions to ALARA be implemented with as much effort as possible to continue improving its design.
